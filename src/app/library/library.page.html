Fullstack Fullstack Library Project React, Redux, TypeScript, C#, PostgreSQL

This project represents the culmination of everything I learned in Integrify's Fullstack course. The backend was built with C# ASP.NET Entity Framework with a PostgreSQL database. The frontend is structured using React Redux with Typescript to create a just-about fully-functioning online Library. It features a structure that should be familiar, allowing the user to browse books by author, categories, and publisher. Users can check their loan history, and edit their profiles.

The bulk of library features are on the admin side. The site features a side container admin panel, only available to authorized users, featuring full CRUD funtionality for all books, and their attributes, as well as processing loans and returns. Please see the site for instructions on how to access the admin features.
