Harmonious HR Project Harmonious HR Task Manager Project Python, Django, PostgreSQL

Harmonious HR Task Manager was our team's entry to the Mimmit Koodaa 2023 Hackathon. Our team consisted of four members, and I worked together with another developer, where we worked very much in tandem, and is my first forray into using Python and Django. The basic bones of the app are that of a todo list. This serves as a demo for what the app intended to be, a proof of concept to show how common HR tasks can be automated, and how the app can be used to track and manage these tasks.

The app features user authentication and authorization, uploading of files, and a basic todo list, and of course the demo of task automation. For more information, please check out the Harmonious HR Task Manager demo video on Youtube.

Feel free check it out on Github and run it locally.
