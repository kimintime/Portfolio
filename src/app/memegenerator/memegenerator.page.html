Memes App Meme Generator Project React, Typescript

The meme generator app does what it says on the page, connect to the memegen.link API, the user can search the most popular memes, add text, and save the meme. If you leave the text input blank, it generates the default meme.

It serves as my second attempt in Typescript, learning my way around defining types and why it matters.

Please feel free to have a try it out, and as always, have a lookover at Github for more, by clicking the buttons below.
