Budget Budget App Project React, TypeScript

This budget app is my first attempt with TypeScript in React. It keeps track of the current balance (income - expenses), giving the user the option to transfer from the balance into savings, and to set target savings, showing the percentage towards that goal. Resetting that target transfers the balance in savings back to the current balance. This allows for a negative current balance, and will alert the user if they try to transfer to savings from a negative current balance.
